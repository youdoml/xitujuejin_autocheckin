# 稀土掘金自动签到脚本

使用Playwright实现的自动化签到脚本，可定时在稀土掘金网站进行每日签到。

## 功能特点

- 自动化签到：使用Playwright模拟浏览器操作完成签到
- 定时执行：支持设置每日定时签到
- Cookie持久化：自动保存和加载登录状态，避免每次都需要重新登录
- 日志记录：详细记录操作过程和结果

## 环境要求

- Python 3.7+
- Playwright
- schedule

## 安装步骤

1. 克隆或下载此项目
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 安装Playwright浏览器：
   ```bash
   playwright install chromium
   ```

## 使用方法

1. 运行脚本：
   ```bash
   python juejin_checkin.py
   ```

2. 首次运行时会打开浏览器窗口，需要手动登录稀土掘金账户
3. 登录成功后，程序会自动保存Cookie信息用于下次自动登录
4. 程序默认会在每天上午9点自动执行签到任务

## 配置说明

- 默认签到时间为每天上午9:00
- Cookie信息保存在 `juejin_cookies.json` 文件中
- 可修改脚本中的时间设置来调整签到时间

## 注意事项

1. 首次运行需要手动登录，后续会自动使用保存的Cookie
2. 如果长时间未使用导致Cookie失效，需要重新登录
3. 请合理使用自动化脚本，避免对服务器造成过大负担
4. 此脚本仅供学习交流使用，请遵守相关法律法规和网站使用条款
